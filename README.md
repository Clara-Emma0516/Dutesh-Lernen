# 🇩🇪 Meine Deutsch-Lernreise | 我的德语学习历程

Willkommen in meinem Repository! 
欢迎来到我的专属代码库！这里主要放了一些语言学习的资料。

备考六级后已经大半年没有学习德语了，一方面是害怕德语会影响英语水平，另一方面可能就是懒吧hh 
语言忘记很容易，想重拾语感却很难，现在脑子里能想起的德语单词已经不超过十个了。
六级分数还没有出来，我也不知道需不需要再考，再刷分。暂且当做这此六级已经高分了吧。
虽然以前不想承认，其实我还是非常享受学习语言的过程的，不管是英语还是德语，所以这个暑假希望能两个语言一起学习吧。
